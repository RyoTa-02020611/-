# Hikari Markets

洗練されたガラスモーフィズムの UI と、サンプルデータで体験できる株式モニタリングアプリです。ウォッチリストの絞り込みやテーマ切り替えなど、インタラクティブな操作をブラウザだけで体験できます。

## セットアップ

1. リポジトリをクローン、またはこのフォルダをダウンロードします。
2. ルートディレクトリで依存関係をインストールします。

   ```bash
   npm install
   ```

3. ローカルサーバーを起動し、ブラウザでアクセスします。

   ```bash
   npm start
   ```

4. `http://localhost:4173` を開き、ウォッチリスト検索やセクターフィルタ、テーマ切り替えボタンなどを操作して UI の挙動を確認してください。

## 主な機能

- ガラスモーフィズムをベースにしたレスポンシブデザイン
- ウォッチリストの検索 & セクターフィルタリング
- モーニングブリーフや AI インサイトなどのカードビュー
- ダーク / ライトテーマのワンタップ切り替え
- 現在時刻のライブ更新とスパークライン描画
- `data/dashboard.json` からマーケットデータをフェッチし、通信失敗時はフォールバックデータに切り替え

## 開発メモ

- 追加のビルドステップは不要です。静的な HTML / CSS / JavaScript で構成されています。
- フォントには Inter（Google Fonts）を使用しています。オフラインの場合はシステムフォントにフォールバックします。
- 表示データを差し替える場合は `data/dashboard.json` を編集するか、API エンドポイントを用意して `script.js` の `fetchJSON` を任意の URL に向けてください。
